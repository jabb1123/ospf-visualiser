General information
  • home page: code.google.com/p/ospf-visualiser
  • application creates its own folder in home dir called .ospf-visualiser which contains file with settings
  • it saves values entered by user, so he dont need to entered the values again

Application tips

Searching

  • use symbol '|' to search more expressions together
  • you can search by IP or name of router
  • you can search subnet which router propagets (for search LLTD net)
  • CTRL+f for focus into search input
Map
  • to display info about router click on it and see on the left side panel
  • to display info about link click on it and see on the left side panel
  • you can see info about mode when you hover with cursor above
  • use shift/ctrl in map
  • you can change shape of the edge in settings
Information panel (on the left side)
  • width of panel is modificable
  • height of routers list is modificable
  • one click on the router to view information about it
